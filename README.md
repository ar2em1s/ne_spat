# NeSpat

## Installation

Install `libxtst` first:

MacOS:
```bash
brew install libxtst
```

Others:
```bash
sudo apt install libxtst-dev
```

Install `ne_spat`:

```bash
gem install ne_spat
```

## Usage

Invoke `ne_spat` from terminal and enjoy:

```bash
ne_spat
```
